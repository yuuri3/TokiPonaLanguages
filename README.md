| P_REMOVE_SOUND          | 音韻変化を起こしたときに音素の脱落が起こる確率                                                                     | 実数   |
| P_MEANING_CHANGE        | 1世代である言語が意味変化を起こす確率                                                                              | 実数   |
| MAX_MEANING_CHANGE_RATE | 意味の最大変化率                                                                                                   | 実数   |
| P_REMOVE_WORD           | 1世代である言語の単語が脱落する確率                                                                                | 実数   |
| P_CREATE_WORD           | 1世代である言語の単語が生成される確率                                                                              | 実数   |
| OLD_TOKI_PONA           | トキポナの単語を記述した.csvファイルのパス<br>単語は縦一列に並べる                                                 | 文字列 |
| PHONETICS               | トキポナ諸語の音素を記述した.csvファイルのパス<br>同じ調音方法の音素は同じ行、同じ調音部位の音素は同じ列に記述する | 文字列 |

| TOKI_PONA_LANGUAGES     | 出力ファイルパス                                                                                                   | 文字列 |

## 仕様概要
* MAP の "0" と記述されたマスにトキポナを配置する。
* MAPのすべてのマスに言語が配置されるまで時間発展させる。
  * 借用
    * 隣接する言語に単語を受け渡しする。
  * 音韻変化
    * 特定の音素1つを脱落させるか、別の音素に変化させる。
    * 変化条件は語頭/語中/語尾から1つを選ぶ。
    * 音節は (C)V(V)(C) に限定する。
* 最終状態を出力ファイルに出力する。
